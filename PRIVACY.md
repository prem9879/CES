# Privacy Policy

## Summary

CES is privacy-first by default.

## Data Handling

- No mandatory PII collection for core execution features.
- Telemetry should remain opt-in.
- Provider keys must remain server-side when server-key mode is enabled.

## Session and Memory

- Session context can be stored in local runtime state.
- MemoryGraph data should be scoped to user/session boundaries.

## Analytics

- Track aggregate operational events only (usage/mode/latency) unless explicit consent is provided.

## Contact

For privacy concerns, open a private security/privacy issue channel.
